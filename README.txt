installer Tesseract lien:
https://github.com/tesseract-ocr/tesseract/releases

Télécharge le fichier nommé tesseract-ocr-w64-setup-xxx.exe (où xxx est la version la plus récente), puis lance-le pour installer Tesseract sur ton PC.

N’oublie pas d’ajouter le dossier d’installation (par défaut : C:\Program Files\Tesseract-OCR) à ta variable d’environnement PATH si ce n’est pas fait automatiquement.
